# Assets Folder

Place the Rod of Asclepius image file here.

**Instructions:**
1. Save your Rod of Asclepius image (PNG, SVG, or JPG) in this folder
2. Name it: `rod-of-asclepius.svg` (or `.png` or `.jpg` depending on format)
3. The image will be available at `/assets/rod-of-asclepius.svg` in the app

Files in the `public` folder are served directly and don't need to be imported.
